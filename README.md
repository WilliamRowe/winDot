# Windows Development Profile

---

Configures PowerShell console environment other development tools with defined configuration files 

## Pre-Req's

---

The following software is required prior to installing the profile :

- PowerShell 7+
- Git


## Install

---

Open a PowerShell instance, and run the following to install the Profile :


```PowerShell
cd $desiredProfilePath

git clone https://WilliamRowe1@dev.azure.com/WilliamRowe1/Profile/_git/Profile .

./setup.ps1

```
